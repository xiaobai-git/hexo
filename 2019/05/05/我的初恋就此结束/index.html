    我的初恋就此结束

    发表于 2019-05-05 更新于 2020-11-18 分类于 感情
    这次你来昆明是吧，你来回机票我出了，可是你为何就不能找下我或者让我找你，我求你给个机会你说不可以...

    嘿嘿，我们分手咯

    难道真的说男生没有钱就不陪谈恋爱吗？难道真的说男生没有钱就不陪谈恋爱吗？

    或许真的是没有钱就不陪谈恋爱，她说去每个地方玩我只让她住廉价的酒店，那是不是也有我不给你买包包口红什么的在内呀？

    对不起啊，我们怎么了？


    这次你来昆明是吧，你来回机票我出了，可是你为何就不能找下我或者让我找你，我求你给个机会你说不可以…不可能，但是你为十么让我知道你来昆明呢？你是为了来看我，还是怎么地，我等了一晚上你的消息为什么一条都没有给我发，不是有句话说女生是书，男生是猪吗?很多时候我也知道你的想法，可是你说你这种暗示我是真的不懂，我知道你脾气不好，我也可以忍受你的脾气，平时吧挺好的，可是最近来几周你是真的变了，变得我越来越不懂了，我不知道要怎么说下去了，唉！
    说实话我也挺喜欢你的，对你的好感特别多，但是从你三月份和我开始闹，说什么分开一段时间，还可以一起出去玩之类的，呵呵，我就想笑，这是什么？然后我问你几年，你却说不要逼你，好，我不逼你。是的，你确实把我逼急了，我接触了情侣空间，但是我知道情侣空间在七天内可以恢复，然后嘞？？直接把我屏蔽了…..这我能说什么，而需要我的时候又找我，好吧我承认你生日时候我贱，我给你买了两百多的生日蛋糕，对我确实贱，清明节放假我回家，我在家好好看视频，你又给我发消息，也就这两天和我聊的特好特好对就是特别好，然后慢慢的对我态度又转变了，呵呵。。你说你给过我机会，是的，确实是，但是我有和你说过我们复合，我想重新向你表白，重新追你，而五一刚好想做这件事，你却这样子对我哈？还有三月份时候我知道你失眠，所以我想陪你聊聊天，每次就说要静静，不想聊天？好吧，你好久都没有好好睡一觉了，我让你去医院你不去，钱我出，你就是死活不去。好嘛，后面你去看了心理医生，看好第一天还好好的，第二天以后就开始抱怨我咯，说我影响到你了，我负能量太多，哪有那么多的负能量啊！！！唉不说了，说了也没什么用，该释怀了，哈哈哈！！噢噢，你说的对，我确实是在做梦做梦去吧、做梦去吧你这个说的不错。

    唉，应该结束了

    或许应该结束了，这是我的初恋，也许你也认为自己还是初恋，随便你了。放假好几天我也喝了好几天的酒了，感觉人也差不多废了，从这篇文章以后我不会再去想你了，你也和我没什么关系了，你也说了嘛，在也不见，哈哈，你也是够绝的，星座都说天蝎座是做事最绝的，但是我做不到

    谢谢你

    谢谢你陪我的五百多天，你说你特别优秀，要求太高，所以我努力变好，要是以后我变好了，变得优秀了，我不要你了怎么办，？？哈哈
    都知道希望有多大，绝望就有多大，这句话说的很对，也就是我爱的太深然后被伤的也深，总之谢谢你，谢谢你……………………………….【就不说了，自己明白】
    以后我对你可能也就那样了……毕竟经历和你的某些事，我害怕了，哈哈

    还有一些话

    以前从来没有人对他说过他是直男，于是他先是查“什么是直男”（真是个直男 ），了解后开始对自己的情商没自信，说话变得小心翼翼；因为自重，从来没说过甜言蜜语的他因为这样的责备变得十分迷茫，才知道自己并不明白惊喜的含义；他拼命刷知乎，想要快速提高情商，却又被指责，于是他听你的话，刷抖音提高情商，结果是看着抖音里别人甜蜜的爱情发酸，一边又自责自己不能给你让别人发酸的爱情。他开始变得担心，暗自下决心给你最好的生活。可他也不是那么的完美，他也想要娱乐。你总觉得他故意很忙，你说他让爱变得很沉重。他精心准备的礼物却总是变成了赔礼道歉。渐渐地，你也许会觉得陪他长大代价太大，他会把从你这里学到的一切对下一个女孩好。“对不起，我不想陪你长大”
